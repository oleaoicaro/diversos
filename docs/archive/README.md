# Archive

Este diretório preserva artefatos históricos do projeto de reposicionamento de carreira do Ícaro Leão. São arquivos de **referência** — não são consumidos por nenhum script do repositório.

| Arquivo | Conteúdo |
|---|---|
| `Historico.txt` | Transcrição da conversa com IA anterior (Fases 1–5) que originou o `cv/` e o `outreach/`. Serve como contexto para sessões futuras. |
| `Profile.pdf` · `Profile.txt` | Exports do perfil LinkedIn original (pré-reescrita). Úteis para auditoria de antes/depois. |
| `report.csv` | Relatório de cursos EY Learning usado para alimentar a seção `ey_learning` em `cv/data/profile.yaml`. |

> Se você chegou aqui procurando o CV vivo: ele está em `cv/` na raiz do repositório. O plano de outreach está em `outreach/`.
