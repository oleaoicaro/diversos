# Archive

Este diretório preserva artefatos históricos do projeto de reposicionamento de carreira do Ícaro Leão. São arquivos de **referência** — não são consumidos por nenhum script do repositório.

| Arquivo | Conteúdo |
|---|---|
| `Historico.txt` | Histórico **curado** da missão de reposicionamento de carreira (Fases 1–6): perfil consolidado, correções acumuladas, checklist do que já está implantado em `cv/`/`outreach/` e backlog do que ainda está em aberto. Fonte de verdade para o próximo agente. |
| `Profile.pdf` · `Profile.txt` | Exports do perfil LinkedIn original (pré-reescrita). Úteis para auditoria de antes/depois. |
| `report.csv` | Relatório de cursos EY Learning usado para alimentar a seção `ey_learning` em `cv/data/profile.yaml`. |
| `cv-melhorias-implementadas.md` | Log da rodada de melhorias do template do CV (anteriormente em `cv/MELHORIAS_IMPLEMENTADAS.md`). |
| `cv-layout-melhorias.md` | Notas de iteração sobre layout / paginação do CV (anteriormente em `cv/LAYOUT_MELHORIAS.md`). |
| `cv-reescrita-executiva-2026.md` | Notas da reescrita executiva 2026 do `profile.yaml` (anteriormente em `cv/REESCRITA_EXECUTIVA_2026.md`). |

> Se você chegou aqui procurando o CV vivo: ele está em `cv/` na raiz do repositório. O plano de outreach está em `outreach/`.
