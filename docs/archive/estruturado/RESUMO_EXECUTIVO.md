## Como Usar Este Documento

### Preparação para Entrevistas
1. Revisar treinamentos da categoria alinhada à vaga
2. Preparar exemplos de aplicação prática
3. Destacar certificações recentes (últimos 2 anos)

### Atualização de CV
1. Consultar `treinamentos_consolidado.csv`
2. Filtrar por categoria relevante
3. Selecionar 5-8 mais impactantes
4. Incluir na seção "Formação Complementar"

### Atualização de LinkedIn
1. Usar `linkedin_highlights.txt` como base
2. Adicionar às "Licenses & Certifications"
3. Mencionar no About: "100+ horas certificadas entre 2022-2026"
4. Destacar categorias: IA, AML, InfoSec, Setorial
