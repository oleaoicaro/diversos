# Outreach — Fase 6

Material de apoio à execução do plano de outreach de 4 semanas (Fase 6 do projeto de reposicionamento de carreira).

## Estrutura

```
outreach/
├── plano_4_semanas.md               # Plano operacional com KPIs semanais
├── templates/
│   ├── 01_connection_request.md     # Pedido de conexão (máx. 300 chars)
│   ├── 02_inmail_pos_aplicacao.md   # Mensagem pós-aplicação
│   ├── 03_followup_7dias.md         # Follow-up D+7
│   └── 04_pos_entrevista.md         # Thank-you note pós-entrevista
└── README.md
```

## Como usar

1. Abrir `plano_4_semanas.md` — é o plano-mãe com metas, cronograma e KPIs.
2. Para cada interação, escolher o template em `templates/` conforme o momento do funil:
   - **Antes de aplicar / descoberta** → `01_connection_request.md`
   - **Logo após aplicar** → `02_inmail_pos_aplicacao.md`
   - **7 dias sem resposta** → `03_followup_7dias.md`
   - **Até 24h após entrevista** → `04_pos_entrevista.md`
3. Cada template traz **4 variantes por ICP**: Fintech BR, Seguradora/Banco tradicional, Fintech Global (em inglês) e um "genérico neutro".
4. Personalizar sempre pelo menos 1 elemento (vaga, produto, post recente do recruiter) antes de enviar.

## ICPs considerados

| ICP | Empresas-alvo no plano | Tom | Idioma |
|---|---|---|---|
| **Fintech BR** | Nubank, Avenue, Mercado Livre, XP | Direto, técnico, product-mindset | PT |
| **Seguradora / Banco tradicional** | Bradesco Seguros, Banco BV | Mais formal, ênfase em regulação | PT |
| **Fintech Global** | Pismo | Técnico, conciso, stack-first | EN |
| **Neutro / Recruiter agência** | Qualquer | Equilibrado | PT ou EN |

## Princípios de estilo (válidos para todos os templates)

- **Máx. 300 caracteres** em connection requests (limite do LinkedIn sem Premium).
- **Máx. 1.300 caracteres** em InMails / mensagens longas.
- **Sempre terminar com um "next step" claro** — uma pergunta simples, uma proposta de conversa de 15 min, ou um recurso para o recruiter consultar.
- **Nunca mentir, nunca inflar** — B1 é B1; 13 seguradoras é 13; projeto pessoal é pessoal.
- **Nenhum cliente EY é nomeado** — sempre "seguradora de Ramos Elementares", "operadora ANS" etc.
